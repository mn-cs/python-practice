# Python Practice Projects

Python exercises covering core concepts, from basic syntax to object-oriented programming, recursion, and GUI development.

## Project Structure

- **1 Introduction to Programming**
- **2 Input, Processing, and Output**
- **3 Decision Structures**
- **4 Repetition Structures**
- **5 Functions**
- **6 Files and Exceptions**
- **7 Lists and Tuples**
- **8 Strings**
- **9 Dictionaries and Sets**
- **10 Classes and OOP**
- **11 Inheritance**
- **12 Recursion**
- **13 GUI Programming**

## How to Run

1. Navigate to a topic folder.
2. Open the `.py` file.
3. Run:

```bash
python3 script_name.py
```

Replace `script_name.py` with the actual name of the Python file (e.g., `python3 car.py`).

## Requirements

- Python 3.x
- Tkinter for GUI exercises
